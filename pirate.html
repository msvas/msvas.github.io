Pirate Puzzle

Find your way through some challenging levels while you try to collect the key to the golden chest. Beat the puzzles to finally become a great pirate!
There are more than 20 challenging levels for you to beat!

This is an Android game made using Game Maker.

This game is not yet available for commercial release, but it is currently in alpha! Send me a message if you want to try the game before everyone else!
